0

开发流程

调证

1.1 笔录信息

  • 平台 - 广州市公安局笔录系统、反诈平台
  • 地址 - 68.32.99.74:91/login
  • 功能
    1. 根据笔录信息，匹配受害人的账号及银行
    2. 登录反诈平台，根据银行卡号，查询归属银行
    3. 案件相关信息写入数据库

1.2 协助查询财产通知书

  • 平台 - 广东省公安厅警综平台新执法办案系统（广州）（新警综）
  • 地址 - 68.26.18.36/index
  • 功能
    1. 登录新警综，填写通知书相关信息

1.3 呈请协助查询财产报告书

  • 平台 - 广东省公安厅警综平台新执法办案系统（广州）（新警综）
  • 地址 - 68.26.18.36/index

1.4 上报

  • 平台 - 国家反诈大数据平台
  • 地址 - fz.xz.ga/index
  • 功能

审批报告（旧）

  • 平台 - 广东省公安厅警综平台新执法办案系统（广州）（新警综）
  • 功能
    1. 登录新警综，填写审批报告相关信息
    2. 制作审批报告书，等待警官手动审批
    3. 审批报告审核通过，得到法律文书

调取证据通知书（旧）

  • 平台 - 广东省公安厅警综平台新执法办案系统（广州）（新警综）
  • 功能
    1. 登录新警综，打印证据通知书文件，制作截图

呈请查询审批表（旧）

  • 平台 - 本地
  • 功能
    1. 根据本地word模版，生成审批表word文件，以及审批表截图

部平台上报（旧）

  • 平台 - 公安部电信诈骗案件侦办平台
  • 表单信息
    • 查询事由模版 - 案件名称，案件编号，现需查询一级账号
