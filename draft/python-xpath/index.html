关于lxml

lxml是python的一个解析库，支持「HTML」和「XML」的解析，支持xpath解析方式，而且解析效率非常高。

XPath，全称XML Path Language，即XML路径语言，它是一门在XML文档中查找信息的语言，它最初是用来搜寻XML文档的，但是它同样适用于HTML文档的搜索。

XPath的选择功能十分强大，它提供了非常简明的路径选择表达式，另外，它还提供了超过100个内建函数，用于字符串、数值、时间的匹配以及节点、序列的处理等，几乎所有我们想要定位的节点，都可以用XPath来选择

XPath于1999年11月16日成为W3C标准，它被设计为供XSLT、XPointer以及其他XML解析软件使用，更多的文档可以访问其官方网站：https://www.w3.org/TR/xpath/

安装

  • Windows/macOS

    pip3 install lxml

  • Linux

    yum install -y epel-release libxslt-devel libxml2-devel openssl-devel
pip3 install lxml

导入

from lxml import etree

  • source - 文件名\路径、文件对象、url
  • parser - 例：etree.XMLParser()、etree.HTMLParser()等，如不设置则使用默认的parser
  • base_url -

etree.xml(text, parser=None, base_url=None) - Parses an XML document or fragment from a string constant.

  • text - 例："<root><test/></root>"
  • parser -
  • base_url -

  • text -
  • parser -
  • base_url -

etree.fromstring(text, parser=None, base_url=None) - Parses an XML document or fragment from a string.

解析对象的一些方法

etree.tostring(element_or_tree, encoding=None, ...) - 解析成字节

etree.tostringlist(element_or_tree, ...) - 解析成列表

xpath方法

使用xpath方法对解析对象进行解析，从而获取节点信息：

parser = etree.HTML(text)
html = parser.xpath(' ') # ' '中的规则如下

xpath规则

表达式 描述
nodename 选取此节点的所有子节点
/ 从根节点选取
// 从匹配选择的当前节点选择文档中的节点，而不考虑它们的位置
. 选取当前节点
.. 选取当前节点的父节点
@ 选取属性

函数

  • starts-with(str) - 获取以str开头的元素

    例：xpath(‘//div[stars-with(@class,”test”)]’)

  • contains() - 包含指定的元素

    获取包含xxx的元素xpath('//div[contains(text,"xxx")]')

    xpath('//div/ul/li[contains(text(),'列表1')]')
xpath('//div/ul/li[contains(@id,'id_name')]')

  • and

与的关系 例子：xpath(‘//div[contains(@id,”test”) and contains(@id,”title”)]’)

  • text()

例子1：xpath(‘//div[contains(text(),”test”)]’) 例子2：xpath(‘//div[@id=”“test]/text()’)
