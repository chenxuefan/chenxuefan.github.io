html 1X

知识点总结

基本结构

<!DOCTYPE html> <!--如果不加，则以怪异模式加载；声明了以下代码将以html5的标准去解析-->
<html>
<head>
  <title></title>
  <style></style>
</head>
<body></body>
  <script></script>
</html>

块级标签

  • h1,h5,div,p,ul,ol,dl,li,dt,dd,hr,pre,iframe,table,th,tr,td,form,header,main,section,article,aside,canvas,audio,video,source,dialog…
  • 特性 - 独占一行

行级标签

  • span,code,a,strong,b,em,i,sup,del,label,small,big
  • 宽度为标签内容的实际宽度，不识别宽高设置
  • 块级和行级相互转换
    • 块转行：display:inline
    • 行转块：display:block;display:inline-block

单标签

  • br,hr,img,input,meta,link…
  • img最好添加alt、width、height属性

双标签

  • 除单标签之外即双标签

html5新增的用于布局的语义化标签

  • header,nav,main,aside,article,section,footer

html5废除的元素

  • center,basefont,big,font,s,strike,tt,u
  • frameset,frame,noframes
  • daoapplet,bgsound,blink,marquee
  • rb(ruby),acronym(abbr),dir(ul),isindex(form/input)

标签属性

  • id,class,title,style,name,width,height,border,align,cellspacing,cellpadding,colcode,row,code,src,href,type,disabled,readonly,placeholder,data-*

表单

button - 可点击按钮（多数情况下，用于通过 JavaScript 启动脚本）

checkbox - 复选框

file - 输入字段和 “浏览"按钮，供文件上传

hidden - 隐藏的输入字段

image - 图像形式的提交按钮

password - 密码字段，该字段中的字符被掩码

radio - 单选按钮

submit - 提交按钮，提交按钮会把表单数据发送到服务器

text - 单行的输入字段，用户可在其中输入文本。默认宽度为 20 个字符
