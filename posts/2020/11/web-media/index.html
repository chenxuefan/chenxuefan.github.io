使用@media（媒体查询）展示及隐藏div

2020-11-19
0
tec
python

目标

当分辨率低了就隐藏，达到某个分辨率后就显示

关键代码

@media (max-height: 600px) {
  #chart{display: none!important;}
}
 
@media (max-width: 300px) {
  #chart{display: none!important;}
}

当max-height，即设备屏幕宽度，小于500px时，id为chart的标签的display值为none，即隐藏该标签，即可达到在手机端隐藏该标签的效果。
