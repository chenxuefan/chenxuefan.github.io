1➕X

php

  • 不是PHP魔术方法 - __autoload()

  • PHP不可以多重继承

  • PHP中，<>表示不等于，和!=的作用一样

  • @符号能够忽略表达式的错误

  • var_dump方法能够输出一个或多个表达式的结构信息

  • mysqli_connect(host,username,password,dbname,port,socket);

  • PHP代码都会嵌入到HTML文档中使用的间隔符有

    <?php ?>   
<script language=＂php＂></script>   
<%  %>

mysql

  • Apache的默认端口是80
  • MySQL的默认端口3306
  • MySQL的进程名为mysqld.exe
  • 字符串类型：char,varchar,blob,text
  • MySQL逻辑备份命令 - mysqldump

JSON

  • JSON 是一种轻量级的数据交换格式
  • JSON 对象由花括号括起来的逗号分割的成员构成
  • JSON 是 JavaScript 对象的字符串表示法
