1+X | 考点小结

2020-11-14
0
tec
1X

html、css、js

  • html5新增了离线缓存机制
  • 使用媒体查询时，媒体类型取值为screen代表 - 用于所有设备
  • range输入类型定义滑块控件
  • border-spacing - 设置相邻单元格的边框间的距离（仅用于“边框分离”模式）
  • border-collapse - 设置表格的边框是否被合并为一个单一的边框
  • 带有默认边框 - filedset、input（无table，td）
  • undefined - var一个变量没有赋值，则它的值为undefined
  • js事件的三个阶段分别为 - 捕获、目标、冒泡
  • a标签的四个伪类 - :link :visited :hover :active
  • background的属性 -
    • background-color
    • background-position
    • background-size
    • background-repeat
    • background-origin
    • background-clip - 设置背景覆盖范围的属性
    • background-attachment
    • background-image
  • <table rules=cols> - 只显示列之间的分隔线
  • relative定位 - 不会脱离文档流，根据元素本身的位置进行定位

php

  • 不是PHP魔术方法 - __autoload()

  • PHP不可以多重继承

  • PHP中，<>表示不等于，和!=的作用一样

  • @符号能够忽略表达式的错误

  • var_dump方法能够输出一个或多个表达式的结构信息

  • mysqli_connect(host,username,password,dbname,port,socket);

  • PHP代码都会嵌入到HTML文档中使用的间隔符有

    <?php ?>   
<script language=＂php＂></script>   
<%  %>

mysql

  • Apache的默认端口是80
  • MySQL的默认端口3306
  • MySQL的进程名为mysqld.exe
  • 字符串类型：char,varchar,blob,text
  • MySQL逻辑备份命令 - mysqldump

bootstrap

  • 可以使用navbar-header类的情况 - 为整个页面添加头部

JSON

  • JSON 是一种轻量级的数据交换格式
  • JSON 对象由花括号括起来的逗号分割的成员构成
  • JSON 是 JavaScript 对象的字符串表示法
  • json() - 不能发起异步请求
