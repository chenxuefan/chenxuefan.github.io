马士基项目开发文档

2021-01-25
0
python RPA

开发流程

一、文件下载

  1. 登录系统，获取cookie值，用于后面的下载请求
  2. 使用js代码
  3. 获取文件下载链接
  4. 按照指定的命名规则下载文件

二、RPA

  1. 打开应用「MODS.bch」
  2. 打开Excel表「Macro.WX-Update and Login」
  3. 操作「Receipt」页
    1. 清空历史数据(5,100)
    2. B列（PO）写入数据
    3. 操作宏按钮「Get Data」
      1. 光标移到最左，确保视图正确
      2. 点击「Get Data」按钮
      3. 点击弹窗上的「START」按钮
    4. T、C、 D列写入数据
  4. 操作「Settled Doc Code」页
    1. 清空历史数据(2,7)
    2. 2到6行写入数据
  5. 操作「Receipt」页
    1. 操作宏按钮「Update Doc」
      1. 点击「Update Doc」按钮
      2. 点击弹窗上的「START」按钮

      1. 为用户部署「达观开发平台」与「所需的python环境」

      2. 部署达观机器人以及上传「license」

        1. 打开桌面的达观机器人

        2. 点击左侧的设置（setting）

        3. 点击中间的许可证（license）

        4. 点击上传许可证（click to upload license），上传的文件名称为“license.txt”，文件的保存路径在下方。

        5. 点击上传证书（click to upload Certificate），上传的文件名称为“Certificate.crt”，文件的保存路径在下方。

        6. 点击激活（activate）,出现“激活成功”字样，则为激活成功

        说明：下方目录下的文件夹命名为用户名称，需要找到对应自己名字的文件夹 路径：H:\GSC\IN\BOM\Commercial\GSC_SCM\SCM\SCM\Walmart and Syngenta\WALMART NEW\CSO LOADINGS\Document\安装文档\license
        （即本说明文档所在路径下的license文件夹）

      1. 遇到很难解决的问题，寻求专业人士解决，不要过于依赖自己的尝试（前提是也要自己尝试找到问题的诱因，以及通过重启或重新运行等措施来排查是否是偶然情况）。提出问题的时候尽量遵循「出现了xxx的问题，我觉得是xxx的原因」
