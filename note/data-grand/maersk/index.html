开发流程

一、文件下载

  1. 登录系统，获取cookie值，用于后面的下载请求
  2. 使用js代码
  3. 获取文件下载链接
  4. 按照指定的命名规则下载文件

二、RPA

  1. 打开应用「MODS.bch」
  2. 打开Excel表「Macro.WX-Update and Login」
  3. 操作「Receipt」页
    1. 清空历史数据(5,100)
    2. B列（PO）写入数据
    3. 操作宏按钮「Get Data」
      1. 光标移到最左，确保视图正确
      2. 点击「Get Data」按钮
      3. 点击弹窗上的「START」按钮
    4. T、C、 D列写入数据
  4. 操作「Settled Doc Code」页
    1. 清空历史数据(2,7)
    2. 2到6行写入数据
  5. 操作「Receipt」页
    1. 操作宏按钮「Update Doc」
      1. 点击「Update Doc」按钮
      2. 点击弹窗上的「START」按钮
