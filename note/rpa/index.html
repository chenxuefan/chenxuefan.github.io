一些必要的知识点

RPA适用场景的特点（工业制造、医疗、教育、银行、保险、证券、信托、房地产、政府职能部门、公检法司等行业领域）

  • 大量重复

    业务流程发生频率高，每日，每周

    高重复性，需定时执行，或手动执行多次

  • 规则明确

    业务流程规则明确，标准化程度高

    涉及系统稳定，更新迭代频次低

  • 准确度高

    跨多个系统操作，不能出错

    涉及结构化，数字化数据处理量大

options.add_experimental_option("prefs",{"safebrowsing.enabled":True})
rpa_brower = Chrome(options)

    rpa_driver = rpa_brower.create(url)

桌面应用

窗口

  • 导入相关的方法和模块

    import rpa.win32._window_action as window

  • 获取窗口句柄 - window._get_hwnd_by_title()

    rpa_window = window._get_hwnd_by_title(win_title)

  • 等待窗口打开 - window.win_appear()

    rpa_window = window.win_appear(win_title,timeout=0)

  • 窗口最大化 - window.win_maximize(win_object/win_title)

  • 窗口最小化 - window.win_minimize(win_object/win_title)

  • 激活窗口 - window.win_active(win_object/win_title)

  • 关闭窗口 - window.win_close(win_object/win_title)

  • 等待窗口关闭 - window.win_disappear(win_object/win_title, timeout=0)

  • 设置窗口位置 - window.set_window_position(win_object, row, col)

    • row - 水平坐标
    • col - 垂直坐标

  • 设置窗口大小 - window.set_window_size(win_object, width, height)

    • width - 窗口宽度
    • height - 窗口高度

界面元素

  • 导入相关的方法和模块

    import rpa.element as element

  • 输入元素文本 - element.input_text()

    element.input_text(window_type="win32",parent=rpa_window,parent_class_type="",child_type="",index=1,count=1,text='')
    • window_type - win32、uia
    • child_type - ComboBox、Button

  • 按钮点击 - element.button_click()

    element.button_click(window_type="uia",parent=rpa_window,parent_class_type="",child_type="",index=1,count=2)

软件自动化

Excel

import rpa.excel

工作簿

  • 打开 - rpa.excel.open(file,visible=True, readonly=False, pass_word="", wps=False, run_type=1)
    • file - 文件名（最好是绝对路径）
    • visible - 是否可见
    • readonly - 是否可读
    • pass_word - 文档密码
    • wps - 打开方式（Office则为False，wps则为True）
    • run_type - 启动方式（1:多实例（默认），0:单实例（用于跨工作簿复制等场景））
  • 创建 - rpa.excel.create(visible=True, wps=False, run_type=1)
  • 激活 - excel_object.activate
    • excel_object - 工作簿对象（open/create）
  • 保存 - excel_object.save(file=os.path.join("./", "fielname"), pass_word="", save_type="xlsx")
    • excel_object - 工作簿对象
    • file - 文件保存路径
    • pass_word - 密码
    • save_type - 文件格式（xlsx/xls…）
  • 关闭 - excel_object.close(save=True)
    • save - 是否默认保存

工作表（sheet）

  • 获取excel对象 - 打开（open）/创建（create）
  • 新建 - excel_object.add_sheet(sheet_name="", location="", relative="after")
    • sheet_name - 新sheet页名称
    • location - 插入位置sheet页名称（可留空格）
    • relative - 插入位置（after：后方，before：前方）
  • 激活 - excel_object.activate_sheet("")
  • 显示 - excel_object.hide_sheet("", is_visible=True)
  • 隐藏 - excel_object.hide_sheet("", is_visible=False)
  • 删除 - excel_object.remove_sheet("")
  • 重命名 - excel_object.rename_sheet("", "")
    • (原名称，新名称)
  • 获取 - excel_object.sheets()[0]
    • 按序号获取，第一个sheet页序号为0
  • 复制 - excel_object.copy_sheet(src_sheet="", new_sheet_name="", another_file="", location="", relative="after")
    • src_sheet - 要复制的sheet页名称，或sheet页对象
    • new_sheet_name - 新sheet页名称
    • another_file - 目标文件的文件名路径，或工作簿对象
    • location - 插入位置sheet名称
    • relative - 插入位置

单元格（cell）

  • 获取sheet对象 - excel_object.get_sheet("")

  • 复制 - sheet_object.copy("A" + str("1"))

    • 单元格 - ("A" + str("1"))
    • 单元格区域 - ("A" + str("1") + ":" + "B" + str("2"))
    • 行 - (str("1"))
    • 列 - ("A")
    • 全部 - ()

  • 清除 - sheet_object.clear("A" + str("1"), "All")

    清除类型

    • “All”：全部
    • “Contents”：内容
    • “Comments”：注释
    • “Formats”：格式
    • “Hyperlinks”：超链接

  • 区域粘贴 - sheet_object.paste_range("A" + str("1"), "Values", "None", False, False)

  • 删除 - sheet_object.delete("A" + str("1"), deleteDirection="up")

    • deletDirection - 方向（up：向上，left：向左）

  • 合并 - sheet_object.merge_cell("A" + str("1") + ":" + "B" + str("2"), each_row=True)

    • each_row - 是否行内合并

  • 插入 - sheet_object.insert("A" + str("1"), insertDirection="down")

    • insertDirection - 方向（down：向下，right：向右）

  • 自动填充 - sheet_object.autofill("", "")

    • 源单元格区域，如A1:B2
    • 目标单元格区域，如A1:B4

  • 写入内容 - sheet_object.write("","")

    • 单元格 - write("A1","text")
    • 单元格区域 - write("A1:b5",iteration)
      • iteration - 可迭代对象，列表或元组

  • 读取内容 - sheet_object.read("")

  • 设置行高 - sheet_object.set_row_height("", 23)

    • 行高 - 设置数值即可，单位为英尺

  • 读取行高 - sheet_object.get_row_height()

  • 设置公式 - sheet_object.set_formula("" "=")

    • 单元格公式（在“=”后面写公式）

  • 读取公式 - excel_object.get_formula("")

  • 设置背景色 - sheet_object.set_style("", "bgcolor", "(0,0,0)")

  • 读取背景色 - sheet_object.get_style("", "bgcolor")

  • 设置字体名称 - sheet_object.set_style("", "fontname", "Times New Roman")

  • 读取字体名称 - sheet_object.get_style("", "fontname")

  • 设置字号 - sheet_object.set_style("", "fontsize", "6")

  • 读取字号 - sheet_object.get_style("", "fontsize")

  • 设置字体颜色 - sheet_object.set_style("", "fontcolor", "(0,0,0)")

  • 读取字体颜色 - sheet_object.get_style("", "fontcolor")

  • 设置注释 - sheet_object.set_comment("", "comment")

  • 读取注释 - sheet_object.get_comment("")

  • 设置数值格式 - sheet_object.set_border_style("", border_style="All", line_style="Continuous", line_weight="Medium", line_color="(0, 0, 0)")

  • 设置对齐方式 - sheet_object.set_style("", style="align", value="Align Left")

    • style="align" - 设置对齐
    • value - 对齐方式
      • Align Left - 左对齐
      • Align Right - 右对齐
      • Center - 居中

  • 设置边框样式

  • 设置粗体 - sheet_object.set_style("", style="bold", value="True")

    • style="bold" - 设置粗体

  • 列去重 - sheet_object.remove_duplicate("A" + str("1") + ":" + "B" + str("2"), ["A","D"])

  • **读取行数 ** - sheet_object.row_count()

  • 读取列数 - sheet_object.col_count()

  • 内容查找 - sheet_object.find("text", ranges="", match_case=True)

    • match_case - 是否区分大小写

  • 内容替换 - sheet_object.replace("old", "new", "")

Word

import rpa.word
  • 打开Word文档 - word = rpa.word.open(file='',visible=True,wps=False)
  • 新建Word文档 - word = rpa.word.create(visible=True,wps=False)
  • 保存Word文档 - word.save(file='')
  • Word转PDF - word.to_pdf(file='')
  • 关闭Word文档 - word.close(save=True)
  • 获取光标当前所在坐标 - coordinate = word.get_position(pattern="cursor")
  • 移动光标 - word.cursor_move(move_type='top', step=1)
    • move_type - top\down\left\right\to_positon\to_start\to_end
  • 设置光标到指定地址 - word.set_cursor(word.address_to_coordinate(page=0,paragraph=0,sentence=0)[1])
    • page - 地址的页数
    • paragraph - 地址的段落序号
    • sentence - 地址的句子序号
  • 写入文本内容 - word.write('', font='Arial', size='12', color='(12,12,12)', bold=False)
