字典排序

  1. 按key
    • methods：sorted(dict)
    • return：包含键名的list（键名已排序）
  2. 按value
    • method：sorted(dict.items(), key = lambda kv:(kv[1], kv[0]))
    • sorted的两个参数，前者是包含数组的list，后者是处理的方法（一个函数）
    • return：包含元组的list（已排序）
