在mac配置ossutil环境

  • 安装 - curl -o ossutilmac64 http://gosspublic.alicdn.com/ossutil/1.7.3/ossutilmac64
  • 赋予权限 - chmod 755 ossutilmac64
  • 改下名 - mv ossutilmac64 ossutil
  • 添加到环境变量 - mv ossutil /usr/local/bin/ossutil
  • 配置 - ./ossutilmac64 config
    • endpoint - oss-cn-beijing.aliyuncs.com
    • accessKeyID - LTAI4GBSAYjB1zdEgFP2ufTN
    • accessKeySecret - ewOC0xhYVEsXAyyO1g2SqAwHE3WSyS
    • stsToken - 好像是可选的
  • 上传文件 - ossutil cp ./ oss://billie-s-blog/ -rf
